# 三种模式快速参考卡

## 一句话总结

| 模式 | 一句话描述 | 推荐指数 |
|------|-----------|---------|
| **Manual** | 我自己决策，不要AI | ⭐⭐⭐ |
| **Assist** | AI给建议，我决定听不听 | ⭐⭐⭐⭐⭐ |
| **Auto** | AI全自动，我只看戏 | ⭐⭐ |

---

## 配置方法

```python
# 在 poker_live_simple.py 文件中修改这一行：

AI_MODE = 'manual'   # 手动模式
AI_MODE = 'assist'   # 辅助模式（推荐）
AI_MODE = 'auto'     # 自动模式
```

---

## 界面对比

### Manual 模式
```
可用行动:
  Bet:
    1. 1/3 Pot (150)
    2. 1/2 Pot (225)
    ...
  7. Check
  8. Fold

请选择行动 (输入数字): _
```

### Assist 模式
```
🤖 AI 正在分析...
💡 AI 建议: Bet 450
📝 理由: 你有顶对，值得下注

可用行动:
  Bet:
    1. 1/3 Pot (150)
    2. 1/2 Pot (225)
    3. 2/3 Pot (300)
    4. Pot (450) 👈 AI推荐    ← 有标记
    ...
  7. Check
  8. Fold
  a. 自动执行AI建议             ← 有快捷键

请选择行动 (输入数字或 a): _
```

### Auto 模式
```
🤖 AI 正在分析...
💡 AI 建议: Bet 450
📝 理由: 你有顶对，值得下注

⚡ 自动模式：执行 AI 建议      ← 自动执行，无需输入

⚡ 执行行动: Bet 450
✓ 行动已执行
```

## 操作方式

| 操作 | Manual | Assist | Auto |
|------|--------|--------|------|
| 输入数字选择 | ✅ | ✅ | ❌ |
| 输入 a 执行AI | ❌ | ✅ | ❌ |
| 自动执行 | ❌ | ❌ | ✅ |
| 查看AI建议 | ❌ | ✅ | ✅ |
| AI标记 | ❌ | ✅ | ✅ |

## 适合人群

### Manual 适合：
- 🎓 初学者（学习基础）
- 💪 想锻炼决策能力的人
- ⚡ 需要快速响应的场景
- 🎯 练习特定策略

### Assist 适合：⭐推荐
- 📚 想学习AI思路的人
- 🎯 日常对局玩家
- 🤔 不确定时需要参考
- 💡 想提升胜率
- **几乎所有人都适合**

### Auto 适合：
- 🔬 测试AI性能
- 📊 收集数据和观察
- 👀 学习AI策略
- 🛠️ 开发调试
- **不适合实际对局**

---

## 速查决策树

```
你想要什么？
│
├─ 我想完全自己决策 
│  → Manual 模式
│
├─ 我想要AI帮助但自己决定
│  → Assist 模式 ⭐推荐
│
└─ 我想让AI自动打
   → Auto 模式
```

---

## 常见场景

### 场景1: 刚开始学习
**推荐**: Manual → Assist
- 先用Manual熟悉操作
- 再用Assist学习策略

### 场景2: 日常对局
**推荐**: Assist
- 大部分时候接受AI建议
- 特殊情况自己判断

### 场景3: 测试AI
**推荐**: Auto
- 观察AI的整体表现
- 记录数据分析

### 场景4: 提升技术
**推荐**: Assist → Manual
- 用Assist学习思路
- 用Manual应用所学

---

## 快捷键

### Manual 模式
- `1-9`: 选择行动
- `0`: 跳过

### Assist 模式
- `1-9`: 选择行动
- `a`: 执行AI建议 ⚡
- `0`: 跳过

### Auto 模式
- 无需输入
- `Ctrl+C`: 退出

---

## 性能对比

```
响应速度:  Manual > Assist > Auto
学习价值:  Assist > Auto = Manual
API消耗:  Manual = 0 | Assist = Auto = 1次/手牌
决策质量:  取决于模式和玩家
```

## 切换模式

**现在运行的是哪个模式？**

启动时会显示：
```
🎮 运行模式: 辅助模式 - AI提供建议
```

**想换模式？**
1. `Ctrl+C` 退出
2. 修改 `poker_live_simple.py` 中的 `AI_MODE`
3. 重新运行 `python poker_live_simple.py`
