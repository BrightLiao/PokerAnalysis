
## 手动使用大模型 chatbot 进行玩家分析
将生成的数据统计指标，和以下 prompt 输入大模型 chatbot，得到玩家分析输出

### 正常 prompt
```
你是一个资深德州扑克教练，上述给出的是一些玩家的德扑数据指标统计，请给这些玩家生成一份分析报告。
分析要求：
 - 总结该玩家的整体打法风格
 - 指出优势和特点
 - 指出明显漏洞和改进建议
 - 提供具体战术建议
 - 与标准打法(如GTO)对比
 - 每日盈利情况是因为打法原因还是运气
  
**输出格式**
首先输出汇总统计表格。
然后按每位玩家输出分析，要求输出全部玩家：
- 玩家姓名
- 风格标签(如"紧凶"、"松凶"、"紧弱"、"松弱")
- 详细分析(200-300字)
- 盈利原因（100-200字）
- 优势清单(3-5条)
- 漏洞清单(3-5条)
- 改进建议(3-5条)
```

### 结合桌面 prompt
```
你是一个资深德州扑克教练，上述给出的是一些玩家的德扑数据指标统计，请给这些玩家生成一份分析报告。
分析要求：

针对整个牌桌风格做分析，并指出这种牌桌风格上最容易盈利的打法
针对每位玩家生成详细分析报告，内容包括但不限于：
总结该玩家的整体打法风格
指出优势和特点
指出明显漏洞和改进建议
提供具体战术建议
与GTO对比以及与桌面最优打法对比
每日盈利情况是因为打法原因还是运气
输出格式
【桌面分析】
对整体桌面的分析
【玩家分析】
按每位玩家输出分析，要求输出全部玩家：
玩家姓名
风格标签(如"紧凶"、"松凶"、"紧弱"、"松弱")
详细分析(200字)
盈利原因（100字）
优势清单(1-3条)
漏洞清单(1-3条)
改进建议(1-3条)

注意：，不用再输出玩家的统计数据表格，只需要生成分析内容
```
