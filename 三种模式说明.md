# 三种运行模式说明

## 模式对比

| 特性 | Manual (手动) | Assist (辅助) | Auto (自动) |
|------|--------------|--------------|-------------|
| AI分析 | ❌ 不使用 | ✅ 使用 | ✅ 使用 |
| AI建议 | ❌ 不显示 | ✅ 显示 | ✅ 显示 |
| AI标记 | ❌ 无 | ✅ 有 | ✅ 有 |
| 预设金额 | ✅ 显示 | ✅ 显示 | ✅ 显示 |
| 用户输入 | ✅ 必需 | ✅ 必需 | ❌ 自动执行 |
| 快捷键 'a' | ❌ 无 | ✅ 有 | ❌ 无需 |
| 适合场景 | 学习/练习 | 日常对局 | 测试/观察 |

---

## 模式 1: Manual（手动模式）

### 特点
- 🎮 完全由玩家决策
- 📊 显示预设下注金额
- ⚡ 无AI延迟
- 🎓 适合学习和练习

### 界面示例

```
可用行动:

  Bet:
    1. 1/3 Pot (150)
    2. 1/2 Pot (225)
    3. 2/3 Pot (300)
    4. Pot (450)
    5. 1.5 Pot (675)
    6. All-in (2500)
  7. Check
  8. Fold
  0. 跳过（不行动）

⏱  [████████████████] 20.0秒

请选择行动 (输入数字): _
```

### 配置方法

```python
# poker_live_simple.py
AI_MODE = 'manual'
```

### 适用场景
- 🎓 学习德州扑克基础
- 💪 锻炼决策能力
- 🎯 练习特定策略
- 🏃 需要快速响应

---

## 模式 2: Assist（辅助模式）⭐推荐

### 特点
- 🤖 AI提供建议
- 👤 玩家最终决策
- 💡 显示决策理由
- 👈 标记推荐选项
- ⚡ 一键执行建议

### 界面示例

```
🤖 AI 正在分析...
💡 AI 建议: Bet 450
📝 理由: 你有顶对，位置好，值得下注获取价值

可用行动:

  Bet:
    1. 1/3 Pot (150)
    2. 1/2 Pot (225)
    3. 2/3 Pot (300)
    4. Pot (450) 👈 AI推荐
    5. 1.5 Pot (675)
    6. All-in (2500)
  7. Check
  8. Fold
  0. 跳过（不行动）
  a. 自动执行AI建议

⏱  [████████████████] 18.5秒

请选择行动 (输入数字或 a): _
```

### 配置方法

```python
# poker_live_simple.py
AI_MODE = 'assist'  # 默认模式
```

### 操作方式

**接受AI建议：**
```
请选择行动 (输入数字或 a): a
⚡ 执行行动: Bet 450
✓ 行动已执行
```

**自己决策：**
```
请选择行动 (输入数字或 a): 7
⚡ 执行行动: Check
✓ 行动已执行
```

**参考AI但选择其他：**
```
AI建议: Bet 450
你选择: 2 (1/2 Pot - 225)
⚡ 执行行动: Bet 225
✓ 行动已执行
```

### 适用场景
- 📚 学习AI的决策逻辑
- 🎯 日常对局辅助决策
- 🤔 不确定时寻求建议
- 💡 提升决策质量
- ⚖️ 平衡学习和胜率

---

## 模式 3: Auto（自动模式）

### 特点
- 🤖 AI自动决策
- ⚡ 自动执行行动
- 👀 仅供观察学习
- 🚀 无需用户输入

### 界面示例

```
🤖 AI 正在分析...
💡 AI 建议: Bet 450
📝 理由: 你有顶对，位置好，值得下注获取价值

⚡ 自动模式：执行 AI 建议

⚡ 执行行动: Bet 450
✓ 行动已执行
```

### 配置方法

```python
# poker_live_simple.py
AI_MODE = 'auto'
```

### 工作流程

1. **检测轮到你** → AI开始分析
2. **AI给出建议** → 显示2秒
3. **自动执行** → 无需输入
4. **等待下一轮** → 循环继续

### 故障处理

**AI分析失败时：**
```
⚠️  AI 分析失败: timeout
⚠️  自动模式下AI失败，使用保守策略
⚡ 执行行动: Check
✓ 行动已执行
```

保守策略：
1. 优先 Check（如果可用）
2. 否则 Fold

### 适用场景
- 🔬 测试AI性能
- 📊 收集数据
- 👀 观察AI策略
- 🛠️ 调试和开发
- 🎓 学习AI思路

### ⚠️ 注意事项

1. **仅供测试** - 不保证获胜
2. **需要监督** - 建议有人观察
3. **API消耗** - 每手牌消耗1次调用
4. **网络依赖** - 需要稳定网络连接
5. **责任自负** - 自动模式风险自担

## 模式切换

### 运行时切换

目前需要修改配置文件重启程序：

```python
# poker_live_simple.py

# 切换到手动模式
AI_MODE = 'manual'

# 切换到辅助模式
AI_MODE = 'assist'

# 切换到自动模式
AI_MODE = 'auto'
```

然后重启：
```bash
python poker_live_simple.py
```

### 命令行参数（高级）

```bash
# 手动模式
python poker_live_client.py "游戏URL" --ai-mode manual

# 辅助模式
python poker_live_client.py "游戏URL" --ai-mode assist

# 自动模式
python poker_live_client.py "游戏URL" --ai-mode auto
```

## 最佳实践

### 新手路径

```
1. Manual模式 (1-2天)
   ↓ 熟悉操作
   
2. Assist模式 (持续使用)
   ↓ 学习AI思路
   
3. Manual模式 (回归)
   ↓ 应用所学
   
4. Assist模式 (精进)
   ↓ 对比差异
```

### 学习建议

**使用 Assist 模式时：**

1. **对比思考**
   - AI建议是什么？
   - 我的想法是什么？
   - 为什么不同？

2. **理解理由**
   - 仔细阅读AI的reasoning
   - 思考背后的逻辑
   - 学习新的视角

3. **记录差异**
   - 记下自己的决策
   - 记下AI的建议
   - 事后分析结果

4. **逐步独立**
   - 先依赖AI
   - 慢慢自己思考
   - 最后验证想法

### 使用 Auto 模式时：

1. **专注观察**
   - 看AI在什么情况下下注
   - 看AI如何控制底池
   - 看AI的位置策略

2. **记录模式**
   - AI的激进程度
   - AI的弃牌频率
   - AI的诈唬时机

3. **分析总结**
   - 一局后总结AI的整体策略
   - 对比自己的打法
   - 找出差异原因

## 常见问题

### Q: 可以中途切换模式吗？
A: 目前需要重启程序。未来版本可能支持热切换。

### Q: Assist模式必须听AI的吗？
A: 不必须。AI只是建议，你可以选择任何行动。

### Q: Auto模式会不会输光？
A: 有可能。AI不保证100%获胜，请谨慎使用。

### Q: Manual模式还有预设金额吗？
A: 有的。预设金额在所有模式都可用。

### Q: 哪个模式最好？
A: **Assist模式**平衡了学习和胜率，推荐日常使用。

### Q: Auto模式适合新手吗？
A: 不太适合。新手应该从Manual或Assist开始，培养决策能力。
